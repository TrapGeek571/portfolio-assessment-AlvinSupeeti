# Portfolio Website
A responsive personal portfolio website built with HTML, CSS, Bootstrap and JavaScript.

## Features
- *Responsive Design*: Works on all devices (desktop, tablet, mobile)
- *Dark Mode Toggle*: Custom theme switching functionality
- *Form Validation*: Client-side form validation with custom alerts
- *Smooth Animations*: CSS animations and transitions
- *Interactive Elements*: DOM manipulation examples throughout
- *Bootstrap Components*: Modern UI components and layout

## Pages
1. *index.html* - Main landing page with:
   - Hero section
   - About me section with "Show More" functionality
   - Skills showcase
   - Featured projects
   - Contact form with validation

2. *projects.html* - Detailed projects page with:
   - Project cards with technology stacks
   - Interactive image slideshow
   - Detailed project information

## Technologies Used
- HTML5
- CSS3 (Grid, Flexbox, Media Queries)
- Bootstrap 5
- JavaScript (ES6+)
- Font Awesome Icons

## JavaScript Features
1. *Theme Toggle*: Dark/light mode switching with localStorage persistence
2. *Form Validation*: Custom validation with Bootstrap classes
3. *Smooth Scrolling*: Animated navigation between sections
4. *Image Slideshow*: Interactive project image viewer
5. *Scroll Animations*: Fade-in effects on scroll
6. *Dynamic Content*: Progress bars and skill ratings
